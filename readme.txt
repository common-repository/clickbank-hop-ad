== Description ==
This plugin will allow you to install a ClickBank text links ad on your site as a 
configurable widget. The widget can then be added via the Design-->Widgets
page in the Admin console.

== Installation ==

1. Upload the the clickbank-hop-ad.zip to your plugins directory and unzip it.
2. Login into your admin site and Activate the ClickBank Hop Ad plugin. (Under Plugins)
3. Go to Design, Widgets
4. Select which sidebar you want to add the widget to.
5. Add the ClickBank Hop Ad widget. By clicking the add link in the title of the ClickBank Hop Ad widget.
6. Then configure the settings, by clicking the Edit link.  Make sure you click Save Settings.
